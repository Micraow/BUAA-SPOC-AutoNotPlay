# BUAA-SPOC-AutoNotPlay

北航SPOC平台刷课脚本，AutoNotPlay有别于AutoPlay版本，适用于服务器网络不佳加载缓慢直接发包改进度。视频play都不play

## 使用方法

打开要刷的课程，定位到具体视频的名称显示出来。按下F12打开开发者工具，进入“网络”，再按下`查看`按钮，应有新请求`addKcnrSfydNew`出现。

右键，复制，“以cURL格式复制” 然后运行脚本，粘贴进去。

按几次回车后输入学号。

选择模式不建议立即快速完成。可以正常模式。视频长度可以看进度条，也可以估一个。倍速随便设，别太离谱，10倍20倍都可。更新时间0.4~2s都行。
